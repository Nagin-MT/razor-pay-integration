# Razor Pay Integration for MERN Stack Apps by Nagin

### This repository contains a MERN (MongoDB, Express.js, React.js, Node.js) stack application integrated with Razor Pay for handling payments.

## Installation Steps

### 1. Server (Node.js)

```bash
    cd server
    npm install
    npm start
```

### 2. Server (Node.js)

```bash
    cd app
    npm install
    npm start
```
